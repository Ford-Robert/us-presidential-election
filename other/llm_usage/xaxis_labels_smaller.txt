4. Flip the Plot Using coord_flip()
If the pollster names are too long or the number of pollsters is large, flipping the axes can often provide better readability:

r
Copy code
ggplot(top_25_pollsters, aes(x = reorder(pollster, n), y = n)) +
  geom_bar(stat = "identity", fill = "skyblue") +
  labs(x = "Pollster", y = "Number of Appearances", title = "Top 25 Pollsters") +
  coord_flip() +  # Flip the x and y axes
  theme(axis.text.y = element_text(size = 8))  # Adjust label size
Flipping the plot will display the pollster names vertically along the y-axis, making it easier to fit long names without cluttering the plot.
